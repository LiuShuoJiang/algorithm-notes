1. Prim算法

2. Kruskal算法(常用)：稀疏图，时间复杂度 $O(m\log m)$

## Kruskal算法

1. 将所有边按权重从小到大排序，本步骤时间复杂度为 $O(m\log m)$；
2. 从小到大依次枚举每条边 $a,b$，权重 $c$，如果 $a,b$ 不连通，就将这条边加入到集合中。本步骤时间复杂度为 $O(m)$。

以上步骤类似于并查集。
