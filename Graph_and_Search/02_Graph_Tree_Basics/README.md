# 树和图的基本知识

树是一种特殊的图，树是无环连通图。

图分为有向图和无向图，无向图是一种特殊的有向图，无向图就是两点之间连着双方向边的有向图。

## 树和图的存储

有向图：

邻接矩阵： $g[a,b]$ ，时间复杂度 $O(n^2)$ 且较费空间
邻接表：类似哈希表的拉链法，每个点都有一个单链表，代表每个点可以走到哪个点，单链表内部点的次序无关紧要
