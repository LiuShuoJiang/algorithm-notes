# 深度优先搜索和广度优先搜索

|     | 数据结构 | 空间   |
|-----|----------|--------|
| DFS | `stack`   | $O(h)$   |
| BFS | `queue`    | $O(2^h)$ |

空间上BFS占劣势，但BFS有“最短路”的性质(若一个图所有边的权重均为1)，而DFS的路径不具有最短性。

## 深度优先搜索DFS

DFS需要考虑**回溯**和**剪枝**。

每个DFS对应一条搜索树，可以把DFS想象成一个“执着的人”。

> 例题：[排列数字](./permutations.cpp)

DFS需要把顺序想清楚。需要注意回溯一定要恢复现场。
